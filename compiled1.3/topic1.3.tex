\section{Exercise 1.3.1}

{\large \textbf{a)} The \textit{floor} function from \(\mathbb{R}\) into \(\mathbb{N}\) is defined by putting \(\lfloor x \rfloor\) to be the largest integer less than or equal to \(x\).
What are the images under the floor function of the following sets?}\\
\([0,1] = \{x \in \mathbb{R} : 0 \leq x \leq 1\}\), \([0,1) = \{x \in \mathbb{R} : 0 \leq x < 1\}\),\\
\((0,1] = \{x \in \mathbb{R} : 0 < x \leq 1\}\), \((0,1) = \{x \in \mathbb{R} : 0 < x < 1\}\)

\[[0,1] = \{x \in \mathbb{R} : 0 \leq x \leq 1\}\]
\[\text{Image} = \{0,1\}\]

\[[0,1) = \{x \in \mathbb{R} : 0 \leq x < 1\}\]
\[\text{Image} = \{0\}\]

\[(0,1] = \{x \in \mathbb{R} : 0 < x \leq 1\}\]
\[\text{Image} = \{0,1\}\]

\[(0,1) = \{x \in \mathbb{R} : 0 < x < 1\}\]
\[\text{Image} = \{0\}\]
